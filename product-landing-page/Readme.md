# SmartX Product Landing Page

A clean, fast, and responsive **static product landing page** built using **HTML5, CSS3, and vanilla JavaScript**.  
This project showcases a modern smartwatch with a focus on performance, accessibility, and good semantic structure.


##  Features

- Fully responsive design (desktop, tablet, mobile)
- Semantic HTML5 structure
- CSS Flexbox & Grid layout
- Scroll animations using AOS library
- Optimized images (WebP) with lazy loading
- Accessible navigation with ARIA labels
- Call-to-action sections for conversions
- Clean and professional UI


##  Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- AOS (Animate On Scroll)
- Remix Icon

##  Accessibility & Performance

- Images include descriptive `alt` attributes
- Icons use `aria-label` for screen readers
- Lazy loading improves page speed
- Optimized assets for better Lighthouse scores


##  Notes

- This is a **static landing page** (no backend)
- Designed as a portfolio / practice project
- Easy to customize for any product
